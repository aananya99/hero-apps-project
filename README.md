# Hero Apps Project

**Project Name:** APP HIVE
**Description:** A modern React-based platform showcasing various apps with detailed information and installation options. Users can view app details, check ratings, install apps, and manage their installed apps.  
**Project URL:** [https://apphiveproject.netlify.app/]

---



## Features
- Browse all apps with cards showing app image, title, and company.
- View app details including description, size, downloads, ratings, and reviews.
- Install apps with a single click and show toast notifications.


## Technologies
- **Frontend:** React, TailwindCSS, DaisyUI, React Router  
- **State & Notifications:** React Hot Toast, React Toastify  
- **Charts:** Recharts  
- **Icons:** FontAwesome  
- **Build Tool:** Vite
